# Практична робота №2

## Обчислення валових викидів шкідливих речовин у атмосферне повітря

Цей проєкт реалізовано на базі фреймворку **Flutter (мова Dart)**. Додаток дозволяє автоматизувати розрахунок викидів твердих частинок (пилу, золи) при спалюванні палива на ТЕС.
### Основні функції:
* **Вугілля:** Розрахунок показника емісії з урахуванням зольності, частки леткої золи та ефективності очистки.
* **Мазут:** Розрахунок викидів згідно з нормативами.
* **Природний газ:** Автоматичне прийняття нульових викидів твердої фази.
